Test Case 1: Successful UPI Transfer via VPA

Test Case ID: UPI_TC_001
Scenario: Verify successful fund transfer using valid personal UPI ID

Preconditions:
- User is logged into UPI-enabled app with a personal UPI ID
- Bank account is linked, active, and has a sufficient balance (e.g., ₹1000)
- User has a valid UPI PIN

Test Steps:
1. Navigate to UPI transfer section
2. Enter a valid personal UPI ID (e.g., example@upi)
3. Enter amount (e.g., ₹100)
4. Optionally, add remarks (e.g., “Test transfer”)
5. Click on “Pay”
6. Enter valid UPI PIN

Expected Result:
- Transaction is successful with a confirmation message: “Transaction successful. Transaction ID: XXXXXXX”
- Transaction ID is generated and displayed
- Balance is deducted correctly (e.g., ₹900 remaining)
- Receiver gets credited instantly
- Transaction is recorded in transaction history

Test Case ID: UPI_TC_002

Preconditions:
- User is logged into UPI-enabled app

Test Steps:
1. Navigate to UPI transfer section
2. Enter a valid UPI ID
